# 22nd Avenue Backend Studio

This is a standalone Sanity Studio for the 22nd Avenue project.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Login to Sanity** (if not already logged in):
    ```bash
    npx sanity login
    ```

3.  **Run the Studio locally**:
    ```bash
    npm run dev
    ```

4.  **Open in your browser**:
    Go to [http://localhost:3333](http://localhost:3333).
